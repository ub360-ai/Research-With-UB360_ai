## üí¨ **Chatting with Professor UB360.ai**

### **Basic Chat:**

1. **Type Your Question**
   - Use the input box at the bottom
   - Ask anything about your uploaded documents
   - Example: "What are the main points in the research paper?"

2. **Select Query Type** (Optional)
   - Answer Question (default)
   - Summarize
   - Compare Sources
   - Extract Key Points
   - Extract Timeline

3. **Send Your Message**
   - Press Enter to send
   - Shift+Enter for a new line
   - Wait for Professor UB360.ai to respond

### **Using @Mentions:**

**Why Use @Mentions?**
- Query specific documents
- More precise answers
- Better context for AI

**How to Use:**
1. Type "@" in the chat input
2. See the autocomplete dropdown appear
3. Type to filter documents (e.g., "@ml")
4. Use arrow keys (‚Üë‚Üì) to navigate
5. Press Enter to select
6. The document name is inserted

**Example:**
```
"What does @machine_learning.pdf say about neural networks?"
```

**Multiple Mentions:**
```
"Compare @doc1.pdf and @doc2.pdf on the topic of AI"
```

### **Understanding Responses:**

**Professor UB360.ai's responses include:**
- **Formatted Text:** Bold, italic, lists
- **Code Blocks:** Syntax-highlighted code
- **Citations:** Sources used for the answer
- **Query Type Badge:** Shows the type of query

**Interacting with Responses:**
- Click "Copy" to copy the response
- View citations to see source documents
- Click source links to see page numbers
